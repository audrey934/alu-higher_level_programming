# Python - Hello World

## Project Overview
This project introduces Python basics. It covers:
- Running Python scripts and inline Python code.
- Printing text, integers, floats, and formatted strings.
- String operations: slicing, concatenation, and repetition.
- Following Python style guidelines (PEP8/pycodestyle).
- Understanding the Zen of Python.

## Files
- `0-run` – Shell script to run a Python file
- `1-run_inline` – Shell script to run inline Python code
- `2-print.py` – Print a simple string
- `3-print_number.py` – Print integer with text
- `4-print_float.py` – Print float with 2 decimals
- `5-print_string.py` – Repeat string 3 times + first 9 chars
- `6-concat.py` – Concatenate strings to form a sentence
- `7-edges.py` – Extract first, last, middle letters of a string
- `8-concat_edges.py` – Concatenate words without new variables
- `9-easter_egg.py` – Print the Zen of Python

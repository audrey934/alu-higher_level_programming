# Python Data Structures Project

Practice **lists, tuples, and sequences** in Python: printing, accessing, replacing, reversing, matrix handling, and tuple operations.  

**Requirements:** Python 3.8.5, executable scripts, no imports, follow pycodestyle.  

**Tasks:** print list, access/replace elements, reverse, copy, remove chars, print matrix, add tuples, string info, find max, divisible by 2, delete, switch values.
